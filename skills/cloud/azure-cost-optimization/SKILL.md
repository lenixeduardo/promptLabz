# Azure Cost Optimisation

Strategies for optimising Azure costs:
- Cost analysis and reporting with Azure Cost Management
- Budget alerts and spending limits
- Reserved instances and savings plans
- Rightsizing underutilised resources
- Storage tier optimisation
- Dev/test pricing and hybrid benefits
