# OpenClaw — Secure Linux Cloud

Hardened Linux cloud server setup:
- Server hardening and minimal attack surface
- Firewall configuration (UFW/iptables)
- SSH key-based authentication and fail2ban
- System monitoring with Prometheus/node_exporter
- Automated backup strategies
- Container runtime security
- Regular security auditing and patching
