# Azure Cloud Migration

Strategies for migrating workloads to Azure:
- Assessment and discovery of on-premises assets
- Migration planning (rehost, refactor, rebuild)
- Azure Migrate tooling and automation
- Database migration with Azure DMS
- Cutover planning and rollback strategies
- Post-migration optimisation and validation
