# Azure Kubernetes

Management of Azure Kubernetes Service (AKS):
- Cluster provisioning and configuration
- Node pool management and scaling
- Deployment strategies (rolling, blue-green, canary)
- Network policies and ingress controllers
- Monitoring with Azure Monitor and Prometheus
- Security and RBAC configuration
