# SEO Audit

Comprehensive SEO auditing:
- Technical SEO analysis (crawlability, indexation, Core Web Vitals)
- On-page SEO optimisation (meta tags, headers, content)
- Off-page SEO (backlink profile, domain authority)
- Content gap analysis
- Structured data and schema markup audit
- Actionable recommendations with priority ranking
