# Subagent-Driven Development

Divide and conquer with specialised sub-agents:
- Architecture agent: Designs the solution structure
- Implementation agent: Writes the code
- Review agent: Validates correctness
- Testing agent: Ensures coverage
- Documentation agent: Writes docs
- Orchestrator: Coordinates and merges results
