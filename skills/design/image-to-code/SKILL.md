# Image to Code

Elite image-to-code workflow:
1. Generates design images specific to each section
2. Deeply analyses composition, spacing, and hierarchy
3. Implements the website to match the design as closely as possible
4. Prefers large, readable, section-specific images over compressed boards
5. Avoids cards-inside-cards-inside-cards UI patterns
