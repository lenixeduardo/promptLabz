# Taste Skill (Premium Design)

An elite taste framework for frontend development that:
- Prevents generic, templated AI-generated designs
- Enforces premium design principles (hierarchy, spacing, typography, colour)
- Applies intentional composition and layout decisions
- Blocks common anti-patterns that make designs look cheap
- Ensures cohesive brand expression across all interfaces
