# Industrial Brutalist UI

Raw mechanical interfaces combining:
- Swiss typographic precision with military terminal aesthetics
- Rigid grid systems and extreme type scale contrast
- Utilitarian colour palettes
- Analog degradation effects and textures
- Optimised for data-heavy dashboards, portfolios, and editorial sites
- Feels like declassified blueprints
