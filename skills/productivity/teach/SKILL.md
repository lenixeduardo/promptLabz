# Teach

Multi-session teaching capability:
- Assesses current user knowledge level
- Creates a structured learning path
- Tracks progress across multiple sessions
- Adapts teaching pace to user comprehension
- Provides practical exercises and examples
- Reinforces learning with spaced repetition
