# Find Skills

Helps users discover agent skills by:
- Analysing user needs and recommending appropriate skills
- Searching the skills.sh directory for relevant skills
- Providing installation instructions for discovered skills
- Categorising skills by domain and author
- Keeping up to date with the latest skill releases
