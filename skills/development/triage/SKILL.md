# Triage

Manages issue triage through a defined state machine:
- **Reporter** — Provides initial context and reproduction steps
- **Responder** — Asks clarifying questions and assigns priority
- **Reviewer** — Validates the fix and closes the loop
- Ensures nothing falls through the cracks
