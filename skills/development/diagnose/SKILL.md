# Diagnose

A systematic debugging workflow:
1. **Reproduce** — Ensure the bug is consistently reproducible
2. **Minimise** — Strip away unrelated code to isolate the root cause
3. **Hypothesise** — Form a theory about what's going wrong
4. **Instrument** — Add logging or debug tooling to confirm
5. **Fix** — Apply the minimal fix
6. **Regression-test** — Verify the fix and ensure no regressions
