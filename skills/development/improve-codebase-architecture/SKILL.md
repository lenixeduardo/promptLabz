# Improve Codebase Architecture

Identifies structural improvements in a codebase by:
- Analysing domain language and bounded contexts
- Detecting coupling and cohesion issues
- Recommending design patterns (e.g., Strategy, Factory, Repository)
- Producing a refactoring roadmap with measurable outcomes
